SAE502 - Migration de données vers ou depuis un environnement NoSQL

Description

Proposition d'une solution de gestion de données NoSQL pour améliorer le suivi des dépenses des clients bancaires, avec une approche visuelle intuitive et une documentation technique détaillée.

Missions réalisées

◆ Etude de l'existant
◆ Maquettage d'interfaces de l'application (IHM)
◆ Modélisation de données
◆ Requêtes NoSQL MongoDB
◆ Rédaction d'un rapport technique et présentatif pour les parties prenantes.

Unités d'enseignements

UE 5.1 - Traiter informatiquement les données à des fins décisionnelles
UE 5.3 - Valoriser les données en contexte professionnel

MongoDB
